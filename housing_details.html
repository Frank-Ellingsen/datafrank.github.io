Housing Price Predictor

Business Problem

Accurately predicting housing prices is critical for buyers, sellers, and real estate professionals. This project builds a machine learning model to support data-driven pricing decisions.

Data & Tools

  • Python (Pandas, NumPy, scikit-learn)
  • Regression models
  • Exploratory Data Analysis (EDA)

Process

  • Data cleaning and preprocessing
  • Feature engineering
  • Model training and evaluation

Results

  • R² score of 0.82
  • RMSE reduced by 18%
  • Key drivers: location, size, number of rooms

Why This Matters

Demonstrates applied machine learning, analytical thinking, and the ability to translate model outputs into business insights.
