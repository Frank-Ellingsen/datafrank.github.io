Actuals vs Budget (Power BI)

Power BI · DAX · Power Query

Project Goal

Turning raw Excel data into actionable business information by implementing a robust financial tracking solution. This project bridges the gap between static budget spreadsheets and dynamic reporting.

The Approach

Financial analysts often spend too much time manually updating budget trackers. I automated this process by creating a seamless data flow from raw exports to a visual dashboard.

Key Features

  • Dynamic DAX Measures: Calculated variance, YTD performance, and forecasting metrics.
  • Data Transformation: Used Power Query to clean and unpivot complex Excel structures.
  • Interactive Visuals: Implemented drill-throughs and bookmarks for an intuitive user experience.

Tools Used

Power BI · DAX · Power Query · Excel
