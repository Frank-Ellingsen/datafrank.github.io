Excel Data Model & Power Pivot

Excel · Power Pivot · Data Modeling ·

Project Scope

Transforming a complex municipal account statement into clear insights using advanced Excel data modeling techniques. This project showcases how Excel can handle sophisticated data processing tasks.

The Solution

Traditional Excel spreadsheets often become slow and difficult to maintain as data grows. I implemented a proper Data Model using Power Pivot to handle large volumes of transactions efficiently.

Key Contributions

  • Data Modeling: Created a star schema within Excel to relate multiple transaction tables.
  • DAX Measures: Developed custom measures for financial aggregation and period-over-period comparison.
  • Power Pivot Integration: Leveraged advanced pivot table features for dynamic reporting.

Tools Used

Excel · Power Pivot · DAX
